This site is an AngularJS Single Page Application, which uses a WebAPI data service provider. With web site design and layout acheived using the Bootstrap 3 front end framework. The site is designed to demonstrate AngularJS programming techniques, third party Javascript controls, HTTP Post opertioans, plus other commonly used modern web site functionality.

Each page has the Javascript source code files used displayed in accordion controls, so the programming techniques can be viewed.
